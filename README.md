# 📊 Document Processing Performance Analytics (Power BI Project)

## Overview
This project showcases a freelance Power BI solution designed to enhance visibility into **document parsing efficiency** and identify **performance bottlenecks**.  
The client already had dashboards for volumes and error monitoring, but lacked insights into **processing time analytics**.  
My task was to design and deliver a **Parsing Time Analytics Dashboard** that transformed raw processing data into actionable business intelligence.

---

## 🔹 Key Features
- **Parsing Time KPIs**: Average, Minimum & Maximum Parse Duration
- **Performance Trends**: Time-based analysis of parsing efficiency
- **Document Type Analysis**: Success rate and duration by document type
- **Company-Wise Comparison**: Benchmarking performance across companies
- **Interactive Drill-Downs**: Decomposition Tree for root cause exploration
- **Dynamic Filtering**: Self-service analytics for stakeholders
